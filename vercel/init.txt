Vercel CLI 32.2.4

  â–² vercel init [example] [dir] [options]

  Initialize example Vercel Projects                                            

  Options:

  -f,  --force  Overwrite destination directory if exists [off]                 


  Global Options:

       --cwd <DIR>            Sets the current working directory for a single   
                              run of a command                                  
  -d,  --debug                Debug mode (default off)                          
  -Q,  --global-config <DIR>  Path to the global `.vercel` directory            
  -h,  --help                 Output usage information                          
  -A,  --local-config <FILE>  Path to the local `vercel.json` file              
       --no-color             No color mode (default off)                       
  -S,  --scope                Set a custom scope                                
  -t,  --token <TOKEN>        Login token                                       
  -v,  --version              Output the version number                         


  Examples:

  - Choose from all available examples

    $ vercel init

  - Initialize example project into a new directory

    $ vercel init <example>

  - Initialize example project into specified directory

    $ vercel <example> <dir>

  - Initialize example project without checking

    $ vercel init <example> --force
