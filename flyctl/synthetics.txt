Synthetic monitoring management.

Usage:
  flyctl synthetics [command]

Available Commands:
  agent       Runs the Synthetics agent

Flags:
  -h, --help   help for synthetics

Global Flags:
  -t, --access-token string   Fly API Access Token
      --debug                 Print additional logs and traces
      --verbose               Verbose output

Use "flyctl synthetics [command] --help" for more information about a command.
