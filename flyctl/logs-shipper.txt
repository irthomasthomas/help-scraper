Manage a VM that ships logs to an external provider

Usage:
  flyctl logs shipper [command]

Available Commands:
  setup       Set up a log shipper VM

Flags:
  -h, --help   help for shipper

Global Flags:
  -t, --access-token string   Fly API Access Token
  -j, --json                  json output
      --verbose               verbose output

Use "flyctl logs shipper [command] --help" for more information about a command.
