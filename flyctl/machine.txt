Commands that manage machines

Usage:
  flyctl machine [command]

Aliases:
  machine, machines, m

Available Commands:
  clone       Clones a Fly Machine
  kill        Kill (SIGKILL) a Fly machine
  list        List Fly machines
  remove      Remove a Fly machine
  run         Run a machine
  start       Start a Fly machine
  status      Show current status of a running mchine
  stop        Stop a Fly machine

Flags:
  -h, --help   help for machine

Global Flags:
  -t, --access-token string   Fly API Access Token
  -j, --json                  json output
      --verbose               verbose output

Use "flyctl machine [command] --help" for more information about a command.
