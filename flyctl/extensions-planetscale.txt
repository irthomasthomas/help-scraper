Provision and manage PlanetScale MySQL databases

Usage:
  flyctl extensions planetscale [command]

Aliases:
  planetscale, mysql

Available Commands:
  create      Provision a PlanetScale MySQL database
  dashboard   Visit the PlanetScale MySQL database dashboard
  destroy     Permanently destroy a PlanetScale MySQL database
  list        List your provisioned PlanetScale MySQL databases
  status      Show details about a PlanetScale MySQL database

Flags:
  -h, --help   help for planetscale

Global Flags:
  -t, --access-token string   Fly API Access Token
      --verbose               Verbose output

Use "flyctl extensions planetscale [command] --help" for more information about a command.
