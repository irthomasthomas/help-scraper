Setup a PlanetScale project for this app

Usage:
  flyctl extensions planetscale [command]

Available Commands:
  create      Provision a PlanetScale project for a Fly.io app
  dashboard   View your PlanetScale database dashboard
  destroy     Permanently destroy a database
  list        List PlanetScale databases for an organization
  status      Show status of a PlanetScale database

Flags:
  -h, --help   help for planetscale

Global Flags:
  -t, --access-token string   Fly API Access Token
      --verbose               Verbose output

Use "flyctl extensions planetscale [command] --help" for more information about a command.
