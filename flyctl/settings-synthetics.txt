Control synthetics agent execution

Usage:
  fly settings synthetics [flags]
  fly settings synthetics [command]

Available Commands:
  disable     Disable synthetics
  enable      Enable synthetics

Flags:
  -h, --help   help for synthetics

Global Flags:
  -t, --access-token string   Fly API Access Token
      --debug                 Print additional logs and traces
      --verbose               Verbose output

Use "fly settings synthetics [command] --help" for more information about a command.
