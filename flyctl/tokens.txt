Manage Fly.io API tokens

Usage:
  flyctl tokens [command]

Available Commands:
  create      Create Fly.io API tokens

Flags:
  -h, --help   help for tokens

Global Flags:
  -t, --access-token string   Fly API Access Token
  -j, --json                  json output
      --verbose               verbose output

Use "flyctl tokens [command] --help" for more information about a command.
