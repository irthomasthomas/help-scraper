Manage users in a postgres cluster

Usage:
  flyctl postgres users [command]

Available Commands:
  list        List users

Flags:
  -h, --help   help for users
  -j, --json   JSON output

Global Flags:
  -t, --access-token string   Fly API Access Token
      --debug                 Print additional logs and traces
      --verbose               Verbose output

Use "flyctl postgres users [command] --help" for more information about a command.
