Create Fly.io API tokens

Usage:
  flyctl tokens create [command]

Available Commands:
  deploy       Create deploy tokens
  litefs-cloud Create LiteFS Cloud tokens
  org          Create org deploy tokens

Flags:
  -h, --help   help for create

Global Flags:
  -t, --access-token string   Fly API Access Token
      --debug                 Print additional logs and traces
      --verbose               Verbose output

Use "flyctl tokens create [command] --help" for more information about a command.
