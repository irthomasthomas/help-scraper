Track major cluster events

Usage:
  flyctl postgres events [command]

Available Commands:
  list        Outputs a formatted list of cluster events

Flags:
  -h, --help   help for events
  -j, --json   JSON output

Global Flags:
  -t, --access-token string   Fly API Access Token
      --debug                 Print additional logs and traces
      --verbose               Verbose output

Use "flyctl postgres events [command] --help" for more information about a command.
