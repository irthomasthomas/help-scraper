"Commands for managing remote builder"

Usage:
  flyctl orgs builder [command]

Available Commands:
  show        Show details about an organization's remote builder image
  update      Update an organization's remote builder image

Flags:
  -h, --help   help for builder

Global Flags:
  -t, --access-token string   Fly API Access Token
  -j, --json                  json output
      --verbose               verbose output

Use "flyctl orgs builder [command] --help" for more information about a command.
