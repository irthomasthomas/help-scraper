Launch and manage Redis clusters managed by Upstash.com

Usage:
  flyctl redis [command]

Available Commands:
  create      Create a new Redis cluster
  delete      Delete a Redis cluster
  list        List Redis clusters for an organization
  plans       List available Redis plans
  status      Show status of a Redis service
  update      Update an Upstash Redis cluster

Flags:
  -h, --help   help for redis

Global Flags:
  -t, --access-token string   Fly API Access Token
  -j, --json                  json output
      --verbose               verbose output

Use "flyctl redis [command] --help" for more information about a command.
