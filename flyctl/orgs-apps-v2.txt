Commands for managing apps v2 default on or off setting for the org

Usage:
  flyctl orgs apps-v2 [command]

Available Commands:
  default-off Default to _not_ using apps v2 for org
  default-on  Default to using apps v2 for org
  show        Show apps v2 default for org

Flags:
  -h, --help   help for apps-v2

Global Flags:
  -t, --access-token string   Fly API Access Token
      --debug                 Print additional logs and traces
      --verbose               Verbose output

Use "flyctl orgs apps-v2 [command] --help" for more information about a command.
