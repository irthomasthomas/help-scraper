Manage health check handlers

Usage:
  flyctl checks handlers [command]

Available Commands:
  create      Create a health check handler
  delete      Delete a health check handler
  list        List health check handlers

Flags:
  -h, --help   help for handlers

Global Flags:
  -t, --access-token string   Fly API Access Token
  -j, --json                  json output
      --verbose               verbose output

Use "flyctl checks handlers [command] --help" for more information about a command.
