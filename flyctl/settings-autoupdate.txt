Control automatic updates

Usage:
  fly settings autoupdate [flags]
  fly settings autoupdate [command]

Available Commands:
  disable     Disable automatic updates
  enable      Enable automatic updates

Flags:
  -h, --help   help for autoupdate

Global Flags:
  -t, --access-token string   Fly API Access Token
      --debug                 Print additional logs and traces
      --verbose               Verbose output

Use "fly settings autoupdate [command] --help" for more information about a command.
