Manage databases in a cluster

Usage:
  flyctl postgres db [command]

Available Commands:
  list        list databases

Flags:
  -h, --help   help for db
  -j, --json   JSON output

Global Flags:
  -t, --access-token string   Fly API Access Token
      --verbose               verbose output

Use "flyctl postgres db [command] --help" for more information about a command.
