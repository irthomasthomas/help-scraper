Manage flyctl settings

Usage:
  flyctl settings [command]

Available Commands:
  analytics   Control anonymized analytics collection
  autoupdate  Control automatic updates
  synthetics  Control synthetics agent execution

Flags:
  -h, --help   help for settings

Global Flags:
  -t, --access-token string   Fly API Access Token
      --debug                 Print additional logs and traces
      --verbose               Verbose output

Use "flyctl settings [command] --help" for more information about a command.
