Help on flyctl commands

Usage:
  flyctl help [flags]
  flyctl help [command]

Available Commands:
  commands    All flyctl commands

Flags:
  -h, --help   help for help

Global Flags:
  -t, --access-token string   Fly API Access Token
      --debug                 Print additional logs and traces
      --verbose               Verbose output

Use "flyctl help [command] --help" for more information about a command.
