Create an API token limited to managing a single app and its resources. Also available as TOKENS DEPLOY

Usage:
  flyctl tokens create deploy [flags]

Flags:
  -a, --app string        Application name
  -x, --expiry duration   The duration that the token will be valid (default 1h0m0s)
  -h, --help              help for deploy
  -n, --name string       Token name (default "flyctl deploy token")

Global Flags:
  -t, --access-token string   Fly API Access Token
  -j, --json                  json output
      --verbose               verbose output
