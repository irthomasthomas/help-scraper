Enable and manage Consul clusters

Usage:
  flyctl consul [command]

Available Commands:
  attach      Attach Consul cluster to an app
  detach      Detach Consul cluster from an app

Flags:
  -h, --help   help for consul

Global Flags:
  -t, --access-token string   Fly API Access Token
      --verbose               verbose output

Use "flyctl consul [command] --help" for more information about a command.
