Shows version information for the flyctl command itself, including version
number and build date.

Usage:
  fly version [flags]
  fly version [command]

Available Commands:
  upgrade      Checks for available updates and automatically upgrades

Flags:
  -h, --help   help for version
  -j, --json   JSON output

Global Flags:
  -t, --access-token string   Fly API Access Token
      --debug                 Print additional logs and traces
      --verbose               Verbose output

Use "fly version [command] --help" for more information about a command.
