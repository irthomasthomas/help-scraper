Shows version information for the flyctl command itself, including version
number and build date.

Usage:
  flyctl version [flags]
  flyctl version [command]

Available Commands:
  update      Checks for available updates and automatically updates

Flags:
  -h, --help                 help for version
  -j, --json                 JSON output
  -s, --saveinstall string   Save parameter in config

Global Flags:
  -t, --access-token string   Fly API Access Token
      --verbose               verbose output

Use "flyctl version [command] --help" for more information about a command.
