Show and manage Postgres configuration.

Usage:
  flyctl postgres config [command]

Available Commands:
  show        Show Postgres configuration
  update      Update Postgres configuration.

Flags:
  -h, --help   help for config

Global Flags:
  -t, --access-token string   Fly API Access Token
      --verbose               verbose output

Use "flyctl postgres config [command] --help" for more information about a command.
