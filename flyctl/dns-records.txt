Manage DNS records within a domain

Usage:
  flyctl dns-records [command]

Available Commands:
  export      Export DNS records
  import      Import DNS records
  list        List DNS records

Flags:
  -h, --help   help for dns-records

Global Flags:
  -t, --access-token string   Fly API Access Token
      --verbose               Verbose output

Use "flyctl dns-records [command] --help" for more information about a command.
