Commands that manage VM instances

Usage:
  flyctl vm [command]

Available Commands:
  restart     Restart a VM
  status      Show a VM's status
  stop        Stop a VM

Flags:
  -h, --help   help for vm

Global Flags:
  -t, --access-token string   Fly API Access Token
      --debug                 Print additional logs and traces
      --verbose               Verbose output

Use "flyctl vm [command] --help" for more information about a command.
