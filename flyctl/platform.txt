The PLATFORM commands are for users looking for information
about the Fly platform.

Usage:
  flyctl platform [command]

Available Commands:
  regions     List regions
  status      Show current platform status with an optional filter for maintenance or incidents in json mode (eg. incidents, maintenance)
  vm-sizes    List VM Sizes

Flags:
  -h, --help   help for platform

Global Flags:
  -t, --access-token string   Fly API Access Token
      --debug                 Print additional logs and traces
      --verbose               Verbose output

Use "flyctl platform [command] --help" for more information about a command.
