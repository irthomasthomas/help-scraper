Open web browser on Fly Web UI for this application

Usage:
  flyctl dashboard [flags]
  flyctl dashboard [command]

Aliases:
  dashboard, dash

Available Commands:
  metrics     Open web browser on Fly Web UI for this app's metrics

Flags:
  -a, --app string      Application name
  -c, --config string   Path to application configuration file
  -h, --help            help for dashboard

Global Flags:
  -t, --access-token string   Fly API Access Token
      --debug                 Print additional logs and traces
      --verbose               Verbose output

Use "flyctl dashboard [command] --help" for more information about a command.
