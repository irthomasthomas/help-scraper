List tokens

Usage:
  flyctl tokens list [flags]

Flags:
  -a, --app string     Application name
  -h, --help           help for list
  -j, --json           JSON output
  -s, --scope string   either 'app' or 'org' (default "app")

Global Flags:
  -t, --access-token string   Fly API Access Token
      --verbose               Verbose output
