V1 APPS ONLY: Autoscaling application resources

Usage:
  flyctl autoscale [command]

Available Commands:
  disable     V1 APPS ONLY: Disable autoscaling
  set         Set app autoscaling parameters
  show        V1 APPS ONLY: Show current autoscaling configuration

Flags:
  -h, --help   help for autoscale

Global Flags:
  -t, --access-token string   Fly API Access Token
      --debug                 Print additional logs and traces
      --verbose               Verbose output

Use "flyctl autoscale [command] --help" for more information about a command.
