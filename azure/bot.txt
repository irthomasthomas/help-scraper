
Group
    az bot : Manage Microsoft Azure Bot Service.

Subgroups:
    authsetting               : Manage OAuth connection settings on a bot.
    directline      [Preview] : Manage the Directline Channel on a bot.
    email           [Preview] : Manage the email Channel on a bot.
    facebook        [Preview] : Manage the Facebook Channel on a bot.
    kik             [Preview] : Manage the Kik Channel on a bot.
    msteams         [Preview] : Manage the Microsoft Teams Channel on a bot.
    skype           [Preview] : Manage the Skype Channel on a bot.
    slack           [Preview] : Manage the Slack Channel on a bot.
    sms             [Preview] : Manage the SMS Channel on a bot.
    telegram        [Preview] : Manage the Telegram Channel on a bot.
    webchat                   : Manage the Webchat Channel on a bot.

Commands:
    create                    : Create a new v4 SDK bot.
    delete                    : Delete an existing bot.
    download        [Preview] : Download an existing bot.
    prepare-deploy            : Add scripts/config files for publishing with `az webapp deployment`.
    prepare-publish [Preview] : (Maintenance mode) Add scripts to your local source code directory
                                to be able to publish back using `az bot publish` for v3 SDK bots.
    publish         [Preview] : Publish to a bot's associated app service.
    show                      : Get an existing bot.
    update                    : Update an existing bot.

To search AI knowledge base for examples, use: az find "az bot"
