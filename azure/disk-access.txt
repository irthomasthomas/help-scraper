
Group
    az disk-access : Manage disk access resources.

Commands:
    create : Create a disk access resource.
    delete : Delete a disk access resource.
    list   : List all the disk access resources under a resource group.
    show   : Get information about a disk access resource.
    update : Update a disk access resource.
    wait   : Place the CLI in a waiting state until a condition is met.

To search AI knowledge base for examples, use: az find "az disk-access"
