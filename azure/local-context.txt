
Group
    az local-context : Manage Local Context.
        WARNING: This command group has been deprecated and will be removed in future release. Use
        'config param-persist' instead.
        WARNING: This command group is experimental and under development. Reference and support
        levels: https://aka.ms/CLI_refstatus

Commands:
    delete : Delete local context data.
    off    : Turn off local context.
    on     : Turn on local context.
    show   : Show local context data.

To search AI knowledge base for examples, use: az find "az local-context"
