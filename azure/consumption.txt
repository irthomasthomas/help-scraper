
Group
    az consumption : Manage consumption of Azure resources.
        WARNING: This command group is in preview and under development. Reference and support
        levels: https://aka.ms/CLI_refstatus
Subgroups:
    budget      : Manage budgets for an Azure subscription.
    marketplace : Inspect the marketplace usage data of an Azure subscription within a billing
                  period.
    pricesheet  : Inspect the price sheet of an Azure subscription within a billing period.
    reservation : Manage reservations for Azure resources.
    usage       : Inspect the usage of Azure resources.

To search AI knowledge base for examples, use: az find "az consumption"
