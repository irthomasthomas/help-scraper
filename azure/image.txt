
Group
    az image : Manage custom virtual machine images.

Subgroups:
    builder : Manage and build image builder templates.

Commands:
    create  : Create a custom Virtual Machine Image from managed disks or snapshots.
    delete  : Delete an Image.
    list    : List the list of images under a resource group.
    show    : Get an image.
    update  : Update custom VM images.
    wait    : Place the CLI in a waiting state until a condition is met.

To search AI knowledge base for examples, use: az find "az image"
