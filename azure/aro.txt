
Group
    az aro : Manage Azure Red Hat OpenShift clusters.

Commands:
    create               : Create a cluster.
    delete               : Delete a cluster.
    get-admin-kubeconfig : List admin kubeconfig of a cluster.
    get-versions         : List versions available for installation.
    list                 : List clusters.
    list-credentials     : List credentials of a cluster.
    show                 : Get the details of a cluster.
    update               : Update a cluster.
    wait                 : Wait for a cluster to reach a desired state.

To search AI knowledge base for examples, use: az find "az aro"
