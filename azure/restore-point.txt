
Group
    az restore-point : Manage restore point with res.

Subgroups:
    collection : Create the restore point collection. Please refer to https://aka.ms/RestorePoints
                 for more details. When updating a restore point collection, only tags may be
                 modified.

Commands:
    create     : Create the restore point. Updating properties of an existing restore point is not
                 allowed.
    delete     : Delete the restore point.
    show       : Get the restore point.
    wait       : Place the CLI in a waiting state until a condition of the restore-point is met.

To search AI knowledge base for examples, use: az find "az restore-point"
