
Group
    az compute-fleet : Manage for Azure Compute Fleet.
        Operations for Azure Compute Fleet.
        WARNING: This command group is in preview and under development. Reference and support
        levels: https://aka.ms/CLI_refstatus

Commands:
    create    : Create an Azure Compute Fleet.
    delete    : Delete an Azure Compute  Fleet.
    list      : List Fleet resources by subscription ID and resource group.
    list-vmss : List VirtualMachineScaleSet resources by Fleet.
    show      : Get a Fleet.
    update    : Update a Fleet.
    wait      : Place the CLI in a waiting state until a condition is met.

To search AI knowledge base for examples, use: az find "az compute-fleet"
