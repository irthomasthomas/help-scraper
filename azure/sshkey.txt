
Group
    az sshkey : Manage ssh public key with vm.

Commands:
    create : Create a new SSH public key resource.
    delete : Delete an SSH public key.
    list   : List all of the SSH public keys.
    show   : Retrieve information about an SSH public key.
    update : Update an SSH public key resource.

To search AI knowledge base for examples, use: az find "az sshkey"
