
Group
    az cdn : Manage Azure Content Delivery Networks (CDNs).

Subgroups:
    custom-domain          : Manage Azure CDN Custom Domains to provide custom host names for
                             endpoints.
    edge-node              : View all available CDN edge nodes.
    endpoint               : Manage CDN endpoints.
    origin                 : List or show existing origins related to CDN endpoints.
    origin-group [Preview] : Manage origin groups of an endpoint.
    profile                : Manage CDN profiles to define an edge network.
    waf                    : Manage CDN WAF.

Commands:
    name-exists            : Check the availability of a resource name. This is needed for resources
                             where name is globally unique, such as a CDN endpoint.
    usage                  : Check the quota and actual usage of the CDN profiles under the given
                             subscription.

To search AI knowledge base for examples, use: az find "az cdn"
