
Group
    az logicapp : Manage logic apps.

Subgroups:
    config     : Configure a logic app.
    deployment : Manage logic app deployments.

Commands:
    create     : Create a logic app.
    delete     : Delete a logic app.
    list       : List logic apps.
    restart    : Restart a logic app.
    scale      : Scale a logic app.
    show       : Get the details of a logic app.
    start      : Start a logic app.
    stop       : Stop a logic app.
    update     : Update a logic app.

To search AI knowledge base for examples, use: az find "az logicapp"
