
Group
    az ts : Manage template specs at subscription or resource group scope.

Commands:
    create : Create a template spec and or template spec version.
    delete : Delete a specified template spec or template spec version by name or resource ID..
    export : Export the specified template spec version and artifacts (if any) to the specified
             output folder.
    list   : List template specs or template spec versions.
    show   : Get the specified template spec or template spec version.
    update : Update a template spec version.

To search AI knowledge base for examples, use: az find "az ts"
