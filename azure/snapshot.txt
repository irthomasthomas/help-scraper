
Group
    az snapshot : Manage point-in-time copies of managed disks, native blobs, or other snapshots.

Commands:
    create        : Create a snapshot.
    delete        : Delete a snapshot.
    grant-access  : Grant read access to a snapshot.
    list          : List snapshots under a resource group.
    revoke-access : Revoke access to a snapshot.
    show          : Get information about a snapshot.
    update        : Update a snapshot.
    wait          : Place the CLI in a waiting state until a condition is met.

To search AI knowledge base for examples, use: az find "az snapshot"
