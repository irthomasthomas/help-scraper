
Group
    az sig : Manage shared image gallery.

Subgroups:
    gallery-application        : Manage gallery application.
    image-definition           : Manage shared gallery image with VM.
    image-version              : Manage shared gallery image version with VM.
    share       [Experimental] : Manage gallery sharing profile.

Commands:
    create                     : Create a shared image gallery.
    delete                     : Delete a Shared Image Gallery.
    list                       : List share image galleries.
    list-shared [Experimental] : List all galleries shared directly to your subscription or tenant
                                 (preview).
    show                       : Retrieves information about a Shared Image Gallery.
    show-shared [Experimental] : Get a gallery that has been shared directly to your subscription or
                                 tenant (preview).
    update                     : Update a share image gallery.

To search AI knowledge base for examples, use: az find "az sig"
