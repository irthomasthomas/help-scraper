
Group
    az hdinsight : Manage HDInsight resources.

Subgroups:
    application                : Manage HDInsight applications.
    autoscale                  : Manage HDInsight cluster's Autoscale configuration.
    azure-monitor              : Manage Azure Monitor logs integration on an HDInsight cluster.
    azure-monitor-agent        : Manage Azure Monitor Agent logs integration on an HDInsight
                                 cluster.
    host                       : Manage HDInsight cluster's virtual hosts.
    monitor                    : Manage Classic Azure Monitor logs integration on an HDInsight
                                 cluster.
    script-action              : Manage HDInsight script actions.

Commands:
    create                     : Create a new cluster.
    delete                     : Deletes the specified HDInsight cluster.
    list                       : List HDInsight clusters in a resource group or subscription.
    list-usage                 : Lists the usages for the specified location.
    resize                     : Resize the specified HDInsight cluster to the specified size.
    rotate-disk-encryption-key : Rotate the disk encryption key of the specified HDInsight cluster.
    show                       : Gets the specified cluster.
    update                     : Update the tags or identity of the specified HDInsight cluster.
    wait                       : Place the CLI in a waiting state until an operation is complete.
