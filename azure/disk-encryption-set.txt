
Group
    az disk-encryption-set : Disk Encryption Set resource.

Subgroups:
    identity                  : Manage identities of a disk encryption set.

Commands:
    create                    : Create a disk encryption set.
    delete                    : Delete a disk encryption set.
    list                      : List disk encryption sets.
    list-associated-resources : Lists all resources that are encrypted with this disk encryption
                                set.
    show                      : Get information of a disk encryption sets.
    update                    : Update a disk encryption set.

To search AI knowledge base for examples, use: az find "az disk-encryption-set"
