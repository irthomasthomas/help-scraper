
Group
    az eventgrid : Manage Azure Event Grid topics, domains, domain topics, system topics partner
    topics, event subscriptions, system topic event subscriptions and partner topic event
    subscriptions.

Subgroups:
    domain             : Manage event domains.
    event-subscription : Manage event subscriptions.
    extension-topic    : Manage Azure Event Grid extension topics.
    partner            : Manage partner resources.
    system-topic       : Manage system topics.
    topic              : Manage Azure Event Grid topics.
    topic-type         : Get details for topic types.

To search AI knowledge base for examples, use: az find "az eventgrid"
