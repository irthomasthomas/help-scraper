
Group
    az feature : Manage resource provider features.

Subgroups:
    registration : Manage resource provider feature registrations.

Commands:
    list         : List preview features.
    register     : Register a preview feature.
    show         : Gets the preview feature with the specified name.
    unregister   : Unregister a preview feature.

To search AI knowledge base for examples, use: az find "az feature"
