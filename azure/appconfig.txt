
Group
    az appconfig : Manage App Configurations.

Subgroups:
    credential : Manage credentials for App Configurations.
    feature    : Manage feature flags stored in an App Configuration.
    identity   : Managed identities for App Configurations.
    kv         : Manage key-values stored in an App Configuration.
    revision   : Manage revisions for key-values stored in an App Configuration.

Commands:
    create     : Create an App Configuration.
    delete     : Delete an App Configuration.
    list       : Lists all App Configurations under the current subscription.
    show       : Show properties of an App Configuration.
    update     : Update an App Configuration.

To search AI knowledge base for examples, use: az find "az appconfig"
