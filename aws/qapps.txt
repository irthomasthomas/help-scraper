NAME
       qapps -

DESCRIPTION
       The  Amazon  Q  Apps feature capability within Amazon Q Business allows
       web experience users to create lightweight, purpose-built  AI  apps  to
       fulfill  specific  tasks from within their web experience. For example,
       users can create an Q Appthat exclusively  generates  marketing-related
       content  to  improve  your marketing team's productivity or a Q App for
       marketing content-generation like writing customer emails and  creating
       promotional content using a certain style of voice, tone, and branding.
       For more information, see Amazon Q App in the Amazon  Q  Business  User
       Guide .
