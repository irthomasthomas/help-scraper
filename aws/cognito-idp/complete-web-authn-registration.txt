NAME
       complete-web-authn-registration -

DESCRIPTION
       Completes registration of a passkey authenticator for the current user.
       Your application provides data from a successful  registration  request
       with the data from the output of a StartWebAuthnRegistration .

       Authorize this action with a signed-in user's access token. It must in-
       clude the scope aws.cognito.signin.user.admin .

       See also: AWS API Documentation

       complete-web-authn-registration uses  document  type  values.  Document
       types  follow the JSON data model where valid values are: strings, num-
       bers, booleans, null, arrays, and objects. For command  input,  options
       and  nested  parameters that are labeled with the type document must be
       provided as JSON. Shorthand syntax does not support document types.
