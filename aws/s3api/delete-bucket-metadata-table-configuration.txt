NAME
       delete-bucket-metadata-table-configuration -

DESCRIPTION
       Deletes  a  metadata table configuration from a general purpose bucket.
       For more information, see Accelerating data discovery with S3  Metadata
       in the Amazon S3 User Guide .
          Permissions

       To  use this operation, you must have the s3:DeleteBucketMetadataTable-
       Configuration permission. For more information, see Setting up  permis-
       sions for configuring metadata tables in the Amazon S3 User Guide .

       The  following  operations are related to DeleteBucketMetadataTableCon-
       figuration :

       o CreateBucketMetadataTableConfiguration

       o GetBucketMetadataTableConfiguration

       See also: AWS API Documentation
