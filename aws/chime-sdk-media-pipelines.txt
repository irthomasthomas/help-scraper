NAME
       chime-sdk-media-pipelines -

DESCRIPTION
       The Amazon Chime SDK media pipeline APIs in this section allow software
       developers to create Amazon Chime SDK  media  pipelines  that  capture,
       concatenate,  or stream your Amazon Chime SDK meetings. For more infor-
       mation about media pipelines, see Amazon Chime SDK media pipelines .
