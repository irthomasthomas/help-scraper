NAME
       workspaces-web -

DESCRIPTION
       Amazon WorkSpaces Secure Browser is a low cost, fully managed WorkSpace
       built  specifically  to   facilitate   secure,   web-based   workloads.
       WorkSpaces Secure Browser makes it easy for customers to safely provide
       their employees with access to internal websites and SaaS web  applica-
       tions  without  the  administrative burden of appliances or specialized
       client software. WorkSpaces Secure Browser provides simple policy tools
       tailored  for user interactions, while offloading common tasks like ca-
       pacity management, scaling, and maintaining browser images.
