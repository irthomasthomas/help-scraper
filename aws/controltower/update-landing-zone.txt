NAME
       update-landing-zone -

DESCRIPTION
       This API call updates the landing zone. It starts an asynchronous oper-
       ation that updates the landing zone based on the new landing zone  ver-
       sion,  or  on  the changed parameters specified in the updated manifest
       file.

       See also: AWS API Documentation

       update-landing-zone uses document type values.  Document  types  follow
       the JSON data model where valid values are: strings, numbers, booleans,
       null, arrays, and objects. For command input, options and nested param-
       eters that are labeled with the type document must be provided as JSON.
       Shorthand syntax does not support document types.
