NAME
       create-sol-function-package -

DESCRIPTION
       Creates a function package.

       A  function package is a .zip file in CSAR (Cloud Service Archive) for-
       mat that contains a network function (an ETSI  standard  telecommunica-
       tion  application)  and function package descriptor that uses the TOSCA
       standard to describe how the network functions should run on your  net-
       work.  For  more  information,  see Function packages in the Amazon Web
       Services Telco Network Builder User Guide .

       Creating a function package is the first step for creating a network in
       AWS  TNB.  This request creates an empty container with an ID. The next
       step is to upload the actual CSAR zip file into that  empty  container.
       To upload function package content, see PutSolFunctionPackageContent .

       See also: AWS API Documentation

SYNOPSIS

          [--cli-input-json <value>]
          [--generate-cli-skeleton <value>]
          [--debug]

          [--no-verify-ssl]
