       NOTE:
          Amazon Q is in preview release and is subject to change.

       This  is  the  Amazon Q (for business use) API Reference. Amazon Q is a
       fully managed, generative-AI powered enterprise chat assistant that you
       can deploy within your organization. Amazon Q enhances employee produc-
       tivity by supporting key tasks such  as  question-answering,  knowledge
       discovery,  writing email messages, summarizing text, drafting document
       outlines, and brainstorming ideas. Users ask questions of Amazon Q  and
       get  answers  that are presented in a conversational manner. For an in-
       troduction to the service, see the ` Amazon Q (for business use) Devel-
       oper                                                              Guide
       https://docs.aws.amazon.com/amazonq/latest/business-use-dg/what-is.html`__
       .

       For  an overview of the Amazon Q APIs, see Overview of Amazon Q API op-
       erations .

       For information about the IAM access control permissions  you  need  to
       use  this API, see IAM roles for Amazon Q in the Amazon Q (for business
       use) Developer Guide .

       You can use the following AWS SDKs to access Amazon Q APIs:

       o AWS SDK for JavaScript

       o AWS SDK for .NET

       o AWS SDK for Python (Boto3)
