NAME
       start-gui-session -

DESCRIPTION
       Initiates a graphical user interface (GUI) session thats used to access
       a virtual computers operating system and application. The session  will
       be  active  for  1 hour. Use this action to resume the session after it
       expires.

       See also: AWS API Documentation

SYNOPSIS

          [--cli-input-json <value>]
          [--generate-cli-skeleton <value>]
          [--debug]
          [--endpoint-url <value>]
          [--no-verify-ssl]
          [--no-paginate]
          [--output <value>]
          [--query <value>]
          [--profile <value>]
          [--region <value>]

          [--no-sign-request]
