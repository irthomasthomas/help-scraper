NAME
       list-ml-model-transform-jobs -

DESCRIPTION
       Returns  a  list of model transform job IDs. See Use a trained model to
       generate new model artifacts .

       When invoking this operation in a Neptune cluster that has IAM  authen-
       tication  enabled,  the IAM user or role making the request must have a
       policy attached that allows the neptune-db:ListMLModelTransformJobs IAM
       action in that cluster.

       See also: AWS API Documentation
