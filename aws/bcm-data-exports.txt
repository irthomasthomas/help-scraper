NAME
       bcm-data-exports -

DESCRIPTION
       You can use the Data Exports API to create customized exports from mul-
       tiple Amazon Web Services cost management and billing datasets, such as
       cost and usage data and cost optimization recommendations.

       The Data Exports API provides the following endpoint:

       o https://bcm-data-exports.us-east-1.api.aws
