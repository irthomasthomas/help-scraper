NAME

DESCRIPTION
       Amazon Web Services Systems Manager is the operations hub for your Ama-
       zon Web Services applications and resources  and  a  secure  end-to-end
       management solution for hybrid cloud environments that enables safe and
       secure operations at scale.

       This reference is intended to be used with the Amazon Web Services Sys-
       tems  Manager  User  Guide  . To get started, see Setting up Amazon Web
       Services Systems Manager .
          Related resources

       o For information about each of the capabilities that comprise  Systems
         Manager,  see Systems Manager capabilities in the Amazon Web Services
         Systems Manager User Guide .

       o For details about predefined runbooks for Automation, a capability of
         Amazon  Web Services Systems Manager, see the * Systems Manager Auto-
         mation runbook reference * .

       o For information about AppConfig, a capability of Systems Manager, see
         the * AppConfig User Guide * and the * AppConfig API Reference * .

       o For  information about Incident Manager, a capability of Systems Man-
         ager, see the * Systems Manager Incident Manager User Guide * and the
         * Systems Manager Incident Manager API Reference * .
