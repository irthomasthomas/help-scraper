NAME
       input-deleted -

DESCRIPTION
       Wait until an input has been deleted It will poll every 5 seconds until
       a successful state has been reached. This will exit with a return  code
       of 255 after 20 failed checks.

       See also: AWS API Documentation

SYNOPSIS

          [--no-paginate]
