NAME
       history -

DESCRIPTION
       Commands  to  interact  with  the  history of AWS CLI commands ran over
       time. To record the history of AWS CLI commands set cli_history to  en-
       abled in the ~/.aws/config file. This can be done by running:

       $ aws configure set cli_history enabled

SYNOPSIS
          history

OPTIONS
       None

AVAILABLE COMMANDS
       o list

       o show
