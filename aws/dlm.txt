NAME
       dlm -

DESCRIPTION
       With  Amazon  Data  Lifecycle  Manager, you can manage the lifecycle of
       your Amazon Web Services  resources.  You  create  lifecycle  policies,
       which are used to automate operations on the specified resources.

       Amazon  Data  Lifecycle  Manager  supports Amazon EBS volumes and snap-
       shots. For information about using Amazon Data Lifecycle  Manager  with
       Amazon  EBS,  see  Amazon Data Lifecycle Manager in the Amazon EC2 User
       Guide .

AVAILABLE COMMANDS
       o create-default-role

       o create-lifecycle-policy

       o delete-lifecycle-policy

       o get-lifecycle-policies

       o get-lifecycle-policy

       o help

       o list-tags-for-resource

       o tag-resource

       o untag-resource

       o update-lifecycle-policy
