NAME
       geo-maps -

DESCRIPTION
       Integrate  high-quality  base  map  data  into  your applications using
       MapLibre . Capabilities include:

       o Access to comprehensive base map data, allowing you to tailor the map
         display to your specific needs.

       o Multiple  pre-designed  map  styles  suited  for  various application
         types, such as navigation, logistics, or data visualization.

       o Generation of static map images for scenarios where interactive  maps
         aren't suitable, such as:

         o Embedding in emails or documents

         o Displaying in low-bandwidth environments

         o Creating printable maps

         o Enhancing application performance by reducing client-side rendering

AVAILABLE COMMANDS
       o get-glyphs

       o get-sprites

       o get-static-map

       o get-style-descriptor

       o get-tile

       o help
