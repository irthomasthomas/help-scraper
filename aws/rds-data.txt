NAME
       rds-data -

DESCRIPTION
       Amazon RDS provides an HTTP endpoint to run SQL statements on an Amazon
       Aurora DB cluster. To run these statements, you use the  RDS  Data  API
       (Data API).

       Data API is available with the following types of Aurora databases:

       o Aurora PostgreSQL - Serverless v2, Serverless v1, and provisioned

       o Aurora MySQL - Serverless v1 only

       For  more information about the Data API, see Using RDS Data API in the
       Amazon Aurora User Guide .

AVAILABLE COMMANDS
       o batch-execute-statement

       o begin-transaction

       o commit-transaction

       o execute-statement

       o help

       o rollback-transaction
