NAME
       resolve-alias -

DESCRIPTION
       Attempts to retrieve a fleet ID that is associated with an alias. Spec-
       ify a unique alias identifier.

       If the alias has a SIMPLE routing strategy, Amazon GameLift  returns  a
       fleet ID. If the alias has a TERMINAL routing strategy, the result is a
       TerminalRoutingStrategyException .
          Related actions

          All APIs by task

       See also: AWS API Documentation
