NAME
       launch-wizard -

DESCRIPTION
       Launch Wizard offers a guided way of sizing, configuring, and deploying
       Amazon Web Services resources for third party applications, such as Mi-
       crosoft  SQL  Server  Always On and HANA based SAP systems, without the
       need to manually identify and provision individual Amazon Web  Services
       resources.

       o delete-deployment

       o get-deployment

       o get-workload

       o help

       o list-deployment-events

       o list-deployments

       o list-workload-deployment-patterns

       o list-workloads
