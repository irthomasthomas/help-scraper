NAME
       list-custom-vocabulary-items -

DESCRIPTION

       tom vocabulary.

       See also: AWS API Documentation

SYNOPSIS

          --bot-version <value>
          --locale-id <value>
          [--max-results <value>]
          [--next-token <value>]

          [--debug]
          [--endpoint-url <value>]

          [--no-paginate]
          [--output <value>]
          [--query <value>]

          [--cli-connect-timeout <value>]
