NAME
       ec2-instance-connect -

DESCRIPTION
       This is the Amazon EC2 Instance Connect API Reference . It provides de-
       scriptions, syntax, and usage examples for each of the actions for Ama-
       zon  EC2  Instance  Connect. Amazon EC2 Instance Connect enables system
       administrators to publish one-time use SSH public keys to EC2,  provid-
       ing users a simple and secure way to connect to their instances.

       To  view the Amazon EC2 Instance Connect content in the Amazon EC2 User
       Guide , see Connect to your Linux instance using EC2 Instance Connect .

       For Amazon EC2 APIs, see the Amazon EC2 API Reference .
