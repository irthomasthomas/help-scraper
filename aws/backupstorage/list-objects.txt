NAME
       list-objects -

DESCRIPTION
       List all Objects in a given Backup.

       See also: AWS API Documentation

       See 'aws help' for descriptions of global parameters.

SYNOPSIS
            list-objects
          --storage-job-id <value>
          [--starting-object-name <value>]
          [--starting-object-prefix <value>]
          [--max-results <value>]
          [--next-token <value>]
          [--created-before <value>]
          [--created-after <value>]
          [--cli-input-json <value>]
          [--generate-cli-skeleton <value>]

OPTIONS
       --storage-job-id (string) Storage job id

       --starting-object-name (string) Optional, specifies the starting Object
       name to list from. Ignored if NextToken is not NULL

       --starting-object-prefix (string) Optional, specifies the starting  Ob-
       ject prefix to list from. Ignored if NextToken is not NULL

       --max-results (integer) Maximum objects count

       --next-token (string) Pagination token

       --created-before (timestamp) (Optional) Created before filter

       --created-after (timestamp) (Optional) Created after filter

       --cli-input-json  (string) Performs service operation based on the JSON
       string provided. The JSON string follows the format provided by  --gen-
       erate-cli-skeleton.  If  other  arguments  are  provided on the command
       line, the CLI values will override the JSON-provided values. It is  not
       possible to pass arbitrary binary values using a JSON-provided value as
       the string will be taken literally.

       --generate-cli-skeleton (string) Prints a  JSON  skeleton  to  standard
       output without sending an API request. If provided with no value or the
       value input, prints a sample input JSON that can be used as an argument
       for  --cli-input-json.  If provided with the value output, it validates
       the command inputs and returns a sample output JSON for that command.

       See 'aws help' for descriptions of global parameters.

OUTPUT
       ObjectList -> (list)
          Object list

          (structure)
              Object

              Name -> (string)
                 Object name

              ChunksCount -> (long)
                 Number of chunks in object

              MetadataString -> (string)
                 Metadata string associated with the Object

              ObjectChecksum -> (string)
                 Object checksum

              ObjectChecksumAlgorithm -> (string)
                 Checksum algorithm

              ObjectToken -> (string)
                 Object token

       NextToken -> (string)
          Pagination token
