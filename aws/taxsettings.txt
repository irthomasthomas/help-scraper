NAME
       taxsettings -

DESCRIPTION
       You  can  use  the tax setting API to programmatically set, modify, and
       delete the tax registration number  (TRN),  associated  business  legal
       name,  and address (Collectively referred to as "TRN information"). You
       can also programmatically view TRN information and tax addresses  ("Tax
       profiles").

       You  can use this API to automate your TRN information settings instead
       of manually using the console.

       Service Endpoint

       o https://tax.us-east-1.amazonaws.com

AVAILABLE COMMANDS
       o batch-delete-tax-registration

       o batch-put-tax-registration

       o delete-supplemental-tax-registration

       o delete-tax-registration

       o get-tax-registration

       o get-tax-registration-document

       o help

       o list-supplemental-tax-registrations

       o list-tax-registrations

       o put-supplemental-tax-registration

       o put-tax-registration
