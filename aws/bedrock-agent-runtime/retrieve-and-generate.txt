       --retrieve-and-generate-configuration (structure)
          Configures the retrieval and generation for the session.

          type -> (string)
              The type of RetrieveAndGenerate.

          knowledgeBaseConfiguration -> (structure)
              Configurations for retrieval and generation for knowledge base.

              knowledgeBaseId -> (string)
                 Identifier of the KnowledgeBase

              modelArn -> (string)
                 Arn of a Bedrock model.

       Shorthand Syntax:

          type=string,knowledgeBaseConfiguration={knowledgeBaseId=string,modelArn=string}

       JSON Syntax:

          {
            "type": "KNOWLEDGE_BASE",
            "knowledgeBaseConfiguration": {
              "knowledgeBaseId": "string",
              "modelArn": "string"
            }
          }

       --session-configuration (structure)
          Configures common parameters of the session.

          kmsKeyArn -> (string)
              The KMS key arn to encrypt the customer data of the session.

       Shorthand Syntax:

          kmsKeyArn=string

       JSON Syntax:

          {
            "kmsKeyArn": "string"
          }
