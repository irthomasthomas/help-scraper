NAME
       scheduler -

DESCRIPTION
       Amazon  EventBridge Scheduler is a serverless scheduler that allows you
       to create, run, and manage tasks from  one  central,  managed  service.
       EventBridge Scheduler delivers your tasks reliably, with built-in mech-
       anisms that adjust your schedules based on the  availability  of  down-
       stream  targets.  The  following  reference lists the available API ac-
       tions, and data types for EventBridge Scheduler.

AVAILABLE COMMANDS
       o create-schedule

       o create-schedule-group

       o delete-schedule

       o delete-schedule-group

       o get-schedule

       o get-schedule-group

       o help

       o list-schedule-groups

       o list-schedules

       o list-tags-for-resource

       o tag-resource

       o untag-resource

       o update-schedule
