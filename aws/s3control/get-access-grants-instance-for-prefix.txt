NAME
       get-access-grants-instance-for-prefix -

DESCRIPTION
       Retrieve  the S3 Access Grants instance that contains a particular pre-
       fix.
          Permissions

       You must have the  s3:GetAccessGrantsInstanceForPrefix  permission  for
       the caller account to use this operation.
          Additional Permissions

       The  prefix  owner  account must grant you the following permissions to
       their S3 Access Grants instance: s3:GetAccessGrantsInstanceForPrefix .

       See also: AWS API Documentation

SYNOPSIS

          --account-id <value>
          --s3-prefix <value>
          [--cli-input-json <value>]
          [--generate-cli-skeleton <value>]
          [--debug]
          [--endpoint-url <value>]
          [--no-verify-ssl]
          [--no-paginate]
          [--output <value>]
          [--query <value>]
